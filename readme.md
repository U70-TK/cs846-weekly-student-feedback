## Structure

In each week directory (e.g., `Week4/`), the following files contain:

- **evaluation.md** - Evaluation of our new guideline
- **example_problems.md** - Example problems
- **guidelines.md** - New guidelines for which our example problems will work
- **feedback.md** - Feedback containing all the above things and the prompts that worked and did not work with GitHub Copilot, along with the outputs
